钓鱼网站的使用、远控

2023/11/28

钓鱼

用root权限打开kali

使用命令更改vim /etc/proxychains4.conf的配置信息

通过更改配置信息使得虚拟机能够使用代理来访问外网

使用命令proxychains4 git clone https://github.com/makdosx/mip22 将mip22下载到当前目录下

随后依次输入

  • ls： 查看是否下载成功
  • cd ./mip22 切换至mip22目录下
  • chmod +x ./* 使当前目录所有用户都具有执行权限
  • proxychains4 bash ./mip22.sh 执行

执行成功

输入1进入

选择一个钓鱼网站，我这里选择【51】Steam

输入3后等待url生成

我这里选择第二个url复制

粘贴至浏览器打开，输入任意数字字母，点击登录

打开mip22文件可以看到两个txt后缀的文件

data.txt存放了刚刚输入的账号密码以及浏览器信息

fingerprints.txt文件里存放了操作系统信息

可以知道通过钓鱼网站可以获取对方的浏览器信息、账号密码及操作系统信息

远控

  • 关闭防火墙以及安全防护

下载“大灰狼”并解压安装

双击运行

点击服务生成，然后将域名ip更改为你的ip地址，然后点击测试

连接成功

点击生成服务后保存

返回安装目录查看

将生成的exe文件拖拽到需要控制的操作系统后双击运行

选中右键对其进行操控
